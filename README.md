# Sudoku Solver
download and use the cmd: python3 SudokuS.py
if loading new board: python3 SudokuS.py new_board.txt

# System requirements
python 3.4 or higher
pygame 2.0.0
SDL 2.0.12 (for MAC)

#Board Requirements
Sudoku board must be in a comma separated fashion as in board.txt
